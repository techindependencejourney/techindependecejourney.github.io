Browsers to use

1: Brave

Brave is a really good browser and it has a built in ad blocker (that doesn't steal your data) and is actually reliable.

We recommend brave because:

  1. It doesn't collect a lot of data
  2. You can import google chrome data if you switch
  3. It is not suspicious (wrote as of 3/20/2023).
  4. It has script blocking
  5. You can block cookies with it.
  6. It supports incognito Tor tabs.
